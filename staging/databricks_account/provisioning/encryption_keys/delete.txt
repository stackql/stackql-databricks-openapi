DELETE
/api/2.0/accounts/{account_id}/customer-managed-keys/{customer_managed_key_id}
Deletes a customer-managed key configuration object for an account. You cannot delete a configuration that is associated with a running workspace.
Path parameters
code: account_id
required
uuid
Databricks account ID of any type. For non-E2 account types, get your account ID from the
Accounts Console
.
code: customer_managed_key_id
required
uuid
Databricks encryption key configuration ID.
Responses
200
The encryption key configuration was successfully deleted.
The encryption key configuration was successfully deleted.
