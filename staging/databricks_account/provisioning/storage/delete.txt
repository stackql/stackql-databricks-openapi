DELETE
/api/2.0/accounts/{account_id}/storage-configurations/{storage_configuration_id}
Deletes a Databricks storage configuration. You cannot delete a storage configuration that is associated with any workspace.
Path parameters
code: account_id
required
uuid
Databricks account ID of any type. For non-E2 account types, get your account ID from the
Accounts Console
.
code: storage_configuration_id
required
uuid
Databricks Account API storage configuration ID.
Responses
200
The storage configuration was successfully deleted.
The storage configuration was successfully deleted.
