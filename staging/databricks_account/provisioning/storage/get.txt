GET
/api/2.0/accounts/{account_id}/storage-configurations/{storage_configuration_id}
Gets a Databricks storage configuration for an account, both specified by ID.
Path parameters
code: account_id
required
uuid
Databricks account ID of any type. For non-E2 account types, get your account ID from the
Accounts Console
.
code: storage_configuration_id
required
uuid
Databricks Account API storage configuration ID.
Responses
200
The storage configuration was successfully returned.
The storage configuration was successfully returned.
code: storage_configuration_id
uuid
Databricks storage configuration ID.
code: storage_configuration_name
string
[ 4 .. 256 ] characters
The human-readable name of the storage configuration.
code: root_bucket_info
object
Root S3 bucket information.
code: bucket_name
string
The name of the S3 bucket.
code: account_id
uuid
The Databricks account ID that hosts the credential.
code: creation_time
int64
Time in epoch milliseconds when the storage configuration was created.
This method might return the following HTTP codes: 401, 404, 500
