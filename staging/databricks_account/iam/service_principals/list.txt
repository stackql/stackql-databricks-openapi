GET
/api/2.0/accounts/{account_id}/scim/v2/ServicePrincipals
Gets the set of service principals associated with a Databricks account.
Path parameters
code: account_id
required
uuid
Databricks account ID
Query parameters
code: filter
string
Query by which the results have to be filtered. Supported operators are equals(
code: eq
), contains(
code: co
), starts with(
code: sw
) and not equals(
code: ne
). Additionally, simple expressions can be formed using logical operators -
code: and
and
code: or
. The
SCIM RFC
has more details but we currently only support simple expressions.
Examples
Summary
code: filter=displayName eq "foo"
Service principals whose displayName equals foo.
code: filter=displayName sw "baz"
Service principals whose displayName starts with baz.
code: filter=displayName co "sp" or displayName co "foo"
Service principals whose displayName contains sp or displayName contains foo.
code: filter=displayName ne "qux"
Service principals whose displayName not equals qux.
code: filter=displayName co "bar"
Service principals whose displayName contains bar.
code: filter=displayName co "sp" and displayName co "foo"
Service principals whose displayName contains sp and displayName contains foo.

string
Comma-separated list of attributes to return in response.
code: excludedAttributes
string
Comma-separated list of attributes to exclude in response.
code: startIndex
int64

code: count
int64
<= 100
Desired number of results per page. Default is 10000.
code: sortBy
string
Attribute to sort the results.
code: sortOrder
string

code: ascending | descending
The order to sort the results.
