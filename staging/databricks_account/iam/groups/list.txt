GET
/api/2.0/accounts/{account_id}/scim/v2/Groups
Gets all details of the groups associated with the Databricks account.
Path parameters
code: account_id
required
uuid
Databricks account ID
Query parameters
code: filter
string
Query by which the results have to be filtered. Supported operators are equals(
code: eq
), contains(
code: co
), starts with(
code: sw
) and not equals(
code: ne
). Additionally, simple expressions can be formed using logical operators -
code: and
and
code: or
. The
SCIM RFC
has more details but we currently only support simple expressions.
Examples
Summary
code: filter=displayName eq "admins"
Groups whose displayName equals admins.
code: filter=displayName sw "bar"
Groups whose displayName starts with bar.
code: filter=displayName co "foo" or displayName co "bar"
Group whose displayName contains foo or displayName contains bar.
code: filter=displayName ne "baz"
Groups whose displayName not equals baz.
code: filter=displayName co "foo"
Groups whose displayName contains foo.
code: filter=displayName co "foo" and displayName co "bar"
Group whose displayName contains foo and displayName contains bar.
code: attributes
string
Comma-separated list of attributes to return in response.
code: excludedAttributes
string
Comma-separated list of attributes to exclude in response.
code: startIndex
int64
Specifies the index of the first result. First item is number 1.
code: count
int64
<= 100
Desired number of results per page. Default is 10000.
code: sortBy
string
Attribute to sort the results.
code: sortOrder
string
Enum:
code: ascending | descending
The order to sort the results.
