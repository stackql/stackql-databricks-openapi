GET
/api/2.0/accounts/{account_id}/metastores/{metastore_id}/storage-credentials
Gets a list of all storage credentials that have been assigned to given metastore.
Path parameters
code: account_id
required
uuid
Databricks account ID of any type. For non-E2 account types, get your account ID from the
Accounts Console
.
code: metastore_id
required
uuid
Unity Catalog metastore ID
Responses
200
The metastore storage credentials were successfully returned.
The metastore storage credentials were successfully returned.
code: storage_credentials
Array of object
An array of metastore storage credentials.
Array [
code: name
string
The credential name. The name must be unique within the metastore.
code: aws_iam_role
object
The AWS IAM role configuration.
code: comment
string
Comment associated with the credential.
code: read_only
boolean
Whether the storage credential is only usable for read operations.
code: owner
string
Username of current owner of credential.
code: full_name
string
The full name of the credential.
code: id
string
The unique identifier of the credential.
code: metastore_id
string
Unique identifier of parent metastore.
code: created_at
int64
Time at which this Credential was created, in epoch milliseconds.
code: created_by
string
Username of credential creator.
code: updated_at
int64
Time at which this credential was last modified, in epoch milliseconds.
code: updated_by
string
Username of user who last modified the credential.
code: used_for_managed_storage
boolean
Whether this credential is the current metastore's root storage credential.
code: isolation_mode
string
Enum:
code: ISOLATION_MODE_OPEN | ISOLATION_MODE_ISOLATED
Whether the current securable is accessible from all workspaces or a specific set of workspaces.
]
