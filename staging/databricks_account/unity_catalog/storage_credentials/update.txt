PUT
/api/2.0/accounts/{account_id}/metastores/{metastore_id}/storage-credentials/{storage_credential_name}
Updates a storage credential on the metastore.
The caller must be the owner of the storage credential. If the caller is a metastore admin, only the
owner
credential can be changed.
Path parameters
code: account_id
required
uuid
Databricks account ID of any type. For non-E2 account types, get your account ID from the
Accounts Console
.
code: metastore_id
required
uuid
Unity Catalog metastore ID
code: storage_credential_name
required
string
Name of the storage credential.
Request body
The storage credential to update.
code: credential_info
object
code: comment
string
Comment associated with the credential.
code: read_only
boolean
Whether the storage credential is only usable for read operations.
code: owner
string
Username of current owner of credential.
code: aws_iam_role
object
The AWS IAM role configuration.
code: isolation_mode
string
Enum:
code: ISOLATION_MODE_OPEN | ISOLATION_MODE_ISOLATED
Whether the current securable is accessible from all workspaces or a specific set of workspaces.
code: new_name
string
New name for the storage credential.
code: skip_validation
boolean
Default
code: false
Supplying true to this argument skips validation of the updated credential.
code: force
boolean
Force update even if there are dependent external locations or external tables.
Responses
201
The storage credential was successfully updated.
The storage credential was successfully updated.
