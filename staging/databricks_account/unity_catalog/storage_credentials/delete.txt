DELETE
/api/2.0/accounts/{account_id}/metastores/{metastore_id}/storage-credentials/{storage_credential_name}
Deletes a storage credential from the metastore. The caller must be an owner of the storage credential.
Path parameters
code: account_id
required
uuid
Databricks account ID of any type. For non-E2 account types, get your account ID from the
Accounts Console
.
code: metastore_id
required
uuid
Unity Catalog metastore ID
code: storage_credential_name
required
string
Name of the storage credential.
Query parameters
code: force
boolean
Force deletion even if the Storage Credential is not empty. Default is false.
Responses
200
The storage credential was successfully deleted.
The storage credential was successfully deleted.
This method might return the following HTTP codes: 401, 404, 500
Error responses are returned in the following format:
{
"error_code"
:
"Error code"
,
"message"
:
"Human-readable error message."
}
