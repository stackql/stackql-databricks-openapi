GET
/api/2.0/accounts/{account_id}/metastores/{metastore_id}/workspaces
Gets a list of all Databricks workspace IDs that have been assigned to given metastore.
Path parameters
code: account_id
required
uuid
Databricks account ID of any type. For non-E2 account types, get your account ID from the
Accounts Console
.
code: metastore_id
required
uuid
Unity Catalog metastore ID
Responses
200
The metastore assignments were successfully returned.
The metastore assignments were successfully returned.
code: workspace_ids
Array of int64
