GET
/api/2.0/accounts/{account_id}/workspaces/{workspace_id}/metastore
Gets the metastore assignment, if any, for the workspace specified by ID. If the workspace
is assigned a metastore, the mappig will be returned. If no metastore is assigned to the
workspace, the assignment will not be found and a 404 returned.
Path parameters
code: account_id
required
uuid
Databricks account ID of any type. For non-E2 account types, get your account ID from the
Accounts Console
.
code: workspace_id
required
int64
Workspace ID.
Responses
200
The workspace metastore assignment was successfully returned.
The workspace metastore assignment was successfully returned.

object

required
string

code: workspace_id
required
int64
The unique ID of the Databricks workspace.
code: default_catalog_name
string
The name of the default catalog in the metastore.
