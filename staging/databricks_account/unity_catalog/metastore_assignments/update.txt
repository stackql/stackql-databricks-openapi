PUT
/api/2.0/accounts/{account_id}/workspaces/{workspace_id}/metastores/{metastore_id}
Updates an assignment to a metastore for a workspace. Currently, only the default catalog
may be updated.
Path parameters
code: account_id
required
uuid
Databricks account ID of any type. For non-E2 account types, get your account ID from the
Accounts Console
.
code: workspace_id
required
int64
Workspace ID.
code: metastore_id
required
uuid
Unity Catalog metastore ID
Request body
The metastore assignment to update.
code: metastore_assignment
object
code: metastore_id
string
The unique ID of the metastore.
code: default_catalog_name
string
Deprecated
The name of the default catalog in the metastore.
This field is depracted. Please use "Default Namespace API" to
configure the default catalog for a Databricks workspace.
Responses
201
The metastore assignment was successfully updated.
The metastore assignment was successfully updated.
