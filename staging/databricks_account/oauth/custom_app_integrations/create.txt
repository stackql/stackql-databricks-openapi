POST
/api/2.0/accounts/{account_id}/oauth2/custom-app-integrations
Create Custom OAuth App Integration.
You can retrieve the custom OAuth app integration via
customappintegration/get
.
Path parameters
code: account_id
required
string
Request body
code: redirect_urls
Array of string
List of OAuth redirect urls
code: name
string
Name of the custom OAuth app
code: confidential
boolean
Default
code: false
This field indicates whether an OAuth client secret is required to authenticate this client.
code: token_access_policy
object
Token access policy
code: access_token_ttl_in_minutes
int32
[ 5 .. 1440 ]
access token time to live in minutes
code: refresh_token_ttl_in_minutes
int32
[ 5 .. 129600 ]
refresh token time to live in minutes
code: scopes
Array of string
OAuth scopes granted to the application.
Supported scopes: all-apis, sql, offline_access, openid, profile, email.
Responses
200
Request completed successfully.
Request completed successfully.
code: integration_id
string
Unique integration id for the custom OAuth app
code: client_id
string
OAuth client-id generated by the Databricks
code: client_secret
string
OAuth client-secret generated by the Databricks.
If this is a confidential OAuth app client-secret will be generated.
This method might return the following HTTP codes: 400, 401, 500, 501
Error responses are returned in the following format:
{
"error_code"
:
"Error code"
,
"message"
:
"Human-readable error message."
}
