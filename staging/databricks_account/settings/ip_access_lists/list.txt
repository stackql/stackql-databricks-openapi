GET
/api/2.0/accounts/{account_id}/ip-access-lists
Gets all IP access lists for the specified account.
Path parameters
code: account_id
required
string
Responses
200
Request completed successfully.
Request completed successfully.
code: ip_access_lists
Array of object
Array [
code: list_id
uuid
Universally unique identifier (UUID) of the IP access list.
code: label
string
Example
code: "Office VPN"
Label for the IP access list. This
cannot
be empty.
code: ip_addresses
Array of string
code: address_count
int32
Example
code: 2
Total number of IP or CIDR values.
code: list_type
string
Enum:
code: ALLOW | BLOCK
Type of IP access list. Valid values are as follows and are case-sensitive:
code: ALLOW
: An allow list. Include this IP or range.
code: BLOCK
: A block list. Exclude this IP or range. IP addresses in the block list are excluded even if they are included in an allow list.
code: created_at
int64
Example
code: 1580265020299
Creation timestamp in milliseconds.
code: created_by
int64
Example
code: 202480738464078
The ID of the user that created this list.
code: updated_at
int64
Example
code: 1580265020299
Update timestamp in milliseconds.
code: updated_by
int64
Example
code: 202480738464089
The ID of the user that last updated this list.
code: enabled
boolean
Specifies whether this IP access list is enabled.
]
